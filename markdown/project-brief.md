# Project Brief: Maintainable Knowledge Asset

## Goal

Create a small reusable knowledge asset that stays easy to edit after import.

## Inputs

- A source article or document.
- A few review notes.
- Any images that explain the structure.

## Decisions

- Keep Markdown as the long-term working format.
- Use headings to show the main sequence.
- Use a Universal Mind Map to scan the structure.

## Open Questions

- Which source details need to remain visible?
- Which details can move to an appendix?
